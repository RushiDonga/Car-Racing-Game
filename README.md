# CarRacingGame 🚗💨

A simple and fun car racing game built using C# and Windows Forms.

---

## 🕹️ How to Run

1. Navigate to the `Release` folder.
2. Double-click on `CarRacingGame.exe` to launch the game.
   
No installation required!
